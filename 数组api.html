数组API

  • forEach 遍历数组所有元素 在foreach中不接受return
  • every 判断数组所有元素是否都符合条件 可以return false中断
  • some 判断数组是否至少有一个元素符合条件 可以return true中断
  • sort 排序
  • map 对数组元素重组 返回一个新的数组
  • filter 过滤符合条件的元素 返回一个新的数组
  • splice 插入或者删除 直接修改当前数组 arr.splice(2,0,"William") 在索引为2上插入元素 arr.splice(2,1,"William") 替换索引为2的元素 arr.splice(2,1) 删除索引为2的元素
  • slice 可从已有的数组中返回选定的元素 不改变原数组
  • unshift() 方法将把它的参数插入 数组的头部
  • shift() 方法从数组中删除第一个元素，并返回该元素的值。此方法更改数组的长度
